# Zomato-Restaurant-Analysis
This project presents an end-to-end restaurant data analysis using Zomato data, focusing on restaurant growth trends, customer behavior patterns, pricing strategies, and digital adoption. Tools used include Excel, SQL, Power BI, and Tableau.

## Business Objectives
- Analyze restaurant presence across countries and cities
- Understand customer ratings and engagement
- Evaluate pricing distribution and performance
- Measure online delivery and table booking adoption
- Identify growth and seasonal trends

---

## Dataset Description
The dataset contains restaurant-level information including:
- Restaurant ID & Name
- Country and City
- Ratings and Votes
- Average Cost for Two
- Online Delivery & Table Booking availability
- Opening Date

---

##  Tools Used & Purpose

###  Excel
- Initial data understanding and validation
- Handling missing values and data consistency
- Exploratory analysis using pivot tables
- Created an Excel-based dashboard for quick insights

###  SQL (MySQL)
- Imported cleaned data into structured tables
- Wrote SQL queries to calculate KPIs
- Used COUNT, AVG, GROUP BY, CASE, and date functions
- Created reusable views for KPI analysis

###  Power BI
- Built proper data model using fact and dimension tables
- Created Date table using DAX
- Designed interactive dashboards with slicers and filters
- Visualized KPIs using cards, charts, and maps

###  Tableau
- Connected dataset and defined table relationships
- Built KPI-wise worksheets
- Created advanced visuals like scatter plots and maps
- Designed interactive dashboards with filters and actions

---

##  KPIs Analyzed
- Number of Restaurants by Country
- Restaurants by City
- Restaurants by Rating
- Online Delivery Percentage
- Table Booking Percentage
- Restaurants by Opening Year / Month / Quarter
- Price Bucket Analysis (Low, Mid, Premium)
- Performance based on Cost and Restaurant Count
- Performance based on Votes and Ratings

---

##  Key Insights
- India has the highest number of restaurants
- Urban cities dominate restaurant presence
- Mid-range pricing is the most common
- Online delivery adoption is higher than table booking
- Restaurants with higher votes generally have higher ratings
- Restaurant growth shows seasonal patterns

---

##  Business Recommendations
- Focus expansion on high-growth countries and cities
- Improve customer experience to boost ratings
- Strengthen online delivery and table booking channels
- Optimize pricing strategies based on regional demand
- Leverage ratings and votes to improve restaurant visibility
